Class: Pales

flag-generator/divisions.Pales(count, gapPercentage, color)

The Pales pattern describes one or more horizontal divisions of the field.

Constructor

new Pales(count, gapPercentage, color)

Instantiates a Pales pattern.
Parameters:
Name Type Description
count number The number of Pales in this instance.
gapPercentage number A whole number representing a percentage of the containerWidth. Used to place gaps during draw time.
color string A hexadecimal color string.
