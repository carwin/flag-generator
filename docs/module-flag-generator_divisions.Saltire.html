Class: Saltire

flag-generator/divisions.Saltire(count, border, borderWidth, color, borderColor)

The Saltire pattern describes two diagonal lines crossing in the center of the field. An X pattern.

Constructor

new Saltire(count, border, borderWidth, color, borderColor)

Creates a Saltire.
Parameters:
Name Type Default Description
count number The number of Saltires in this instance. Should typically only be one.
border boolean false A boolean value to decide whether or not to draw a border based on the global seed setting.
borderWidth number 0 A number value to use when generating border width. This number is used in addition to the randomly generated border width determined by the seed.
color string A hexadecimal color string.
borderColor string A hexadecimal color string.
Source:
  • divisions.js, line 282
Example
// Instantiates a Saltire without a border.
const saltire = new Saltire();
// Instantiates a Saltire with a border.
const saltireWithBorder = new Saltire(true);
